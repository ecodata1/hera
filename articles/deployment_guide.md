### Quick Start

To add a new assessment to `hera` package install `hera` package

    install.packages("devtools")
    devtools::install_github("aquaMetrics/hera", dependencies = TRUE)
    library(hera)

1.  Open a new File \> New File \> R Markdown
2.  Select ‘From Template’
3.  Search for ‘Deployment Template’ and open
4.  Change the ‘Deployment Template’ title to match your assessment name
5.  Save the document

Congratulations! You now have a new deployment template ready to
populate.
