# Launch hera app

Launches hera shiny app with optional data and \`catalogue\`.

## Usage

``` r
launch_app(new_catalogue = NULL, data = NULL)
```

## Arguments

- new_catalogue:

  Dataframe of catalogue see \`catalogue\`

- data:

  Dataframe of variables in WFD inter-change format

## Value

Shiny app

## Examples

``` r
if (FALSE) { # \dontrun{
launch_app()
} # }
```
