# Catalogue of assessments

A data frame containing descriptions, data and functions used by
assessment methods. This catalogue is automatically extended by knitting
new assessment vignettes and rebuilding the package. See Development
Guide on package website.

## Usage

``` r
catalogue
```

## Format

A data frame with 3 variables:

- assessment:

  Name of assessment

- data:

  List data frame describing the input and output data of the assessment

- assessment_function:

  Assessment function that generates calculated output values

## Source

See each assessment for detailed references.
